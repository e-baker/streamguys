This plugin allows you to add the StreamGuys player via a shortcode into any page/post.

== Description ==

The StreamGuys Player Plugin allows you to have the StreamGuys player show on a post or page without having to copy and paste the proper code each and every time. All you need to do is install the plugin, add your API key, and put the shortcode on whichever page you wish for the player to show.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/streamguys` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->StreamGuys screen to configure the plugin with your API key
4. Add the `[streamguys]` shortcode to a page or post in which you want the player to show.


== Frequently Asked Questions ==

= Do I need to have a StreamGuys account first? =

Yes, you will need an API key in order to have this plugin work.
